# MIND

Next.js (App Router) : page avec header + menu (ancres), hero, trois sections, footer, titre **MIND**. API pour le backend.

## Développement

```bash
npm install
npm run dev
```

URL locale : `http://localhost:8001` (port défini dans `package.json`).

## API

- `GET /api/health` → `{ "ok": true, "app": "MIND" }`

## Build

```bash
npm run build
npm start
```
